Simple and additional colors

To use simple and additional colors you must add special prefix to color name.
Example: bg-* - for background, fg-* - for text color. In table below specified all prefixes.

focus bg-*-focus, fg-*-focus, bd-*-focus, ol-*-focus
ribbed background ribbed-*
opacity background op-*
